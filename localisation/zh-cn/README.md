# .NET MCP 工作坊

您是否有兴趣构建 MCP 服务器？MCP 客户端怎么样？您希望在哪里运行 MCP 服务器 - 本地主机还是 Azure？让我们一起构建和部署它们！

## 工作坊目标

- 以两种不同的方式构建待办事项列表 MCP 服务器。
- 构建 Blazor Web 应用作为 MCP 客户端。
- 容器化 MCP 服务器。
- 在本地和 Azure 上远程运行 MCP 服务器。
- 将 MCP 服务器部署到 Azure Container Apps。
