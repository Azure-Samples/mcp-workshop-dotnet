# 开始使用

欢迎来到 .NET MCP 工作坊！这是您 MCP 应用开发之旅的起点。

此目录包含您的 MCP 服务器项目的初始结构。您将使用它作为基础来构建一个可以在本地和 Azure 上运行的待办事项列表管理 MCP 服务器。

## 接下来做什么？

按照工作坊说明进行：

1. 设置您的开发环境
1. 构建您的 MCP 服务器
1. 部署到 Azure
1. 创建 MCP 客户端

如果您想查看完整示例，请查看 [/complete](../complete) 目录。
