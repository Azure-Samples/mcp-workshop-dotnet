# 시작하기

.NET용 MCP 워크샵에 오신 것을 환영합니다! 이것은 MCP 앱 개발 여정의 시작점입니다.

이 디렉토리에는 MCP 서버 프로젝트의 초기 구조가 포함되어 있습니다. 이를 기반으로 로컬과 Azure 모두에서 실행할 수 있는 할 일 목록 관리 MCP 서버를 구축할 것입니다.

## 다음 단계는?

워크샵 지침을 따라:

1. 개발 환경 설정
1. MCP 서버 구축
1. Azure에 배포
1. MCP 클라이언트 생성

완전한 예제를 보고 싶다면 [/complete](../complete) 디렉토리를 확인하세요.
